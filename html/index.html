Let's Eat is a full stack MEAN MVC application where users can read or create restaurant reviews. The front end functionality is built using Angular, written in TypeScript, while the styling is done using Bootstrap. The back end logic is written in JavaScript using the Express framework and the MongoDB non-relational database. The application implements a RESTful API interface between the client and the database and utilizes both front end and back end validations to preserve the integrity of the data. It is deployed on an Ubuntu 18.04 Azure Virtual Machine. HTTP requests to the application site are routed by an Nginx web server to a Node.js server that interfaces directly with the application.

Technologies used:

  • Angular
  • Express
  • MongoDB
  • Node.js
  • Azure VM
  • Nginx
  • Bootstrap
  • JavaScript
  • TypeScript

Activity Center is a full stack ASP.NET Core MVC application where users can list or sign up to participate in community events and activities. The front end component is built using HTML, CSS and Bootstrap. The back end logic is written in C# using Entity Framework Core to interface with the MySQL database. The application uses back-end validations to preserve the integrity of the data being stored in the database. It is deployed on an Ubuntu 18.04 Azure Virtual Machine. HTTP requests to the application site are routed by an Nginx web server to a Kestrel server that's included with the ASP.NET core framework and interfaces with the application.

Technologies used:

  • ASP.NET Core
  • Entity Framework Core
  • MySQL
  • Azure VM
  • Nginx
  • Bootstrap
  • C#
  • HTML
  • CSS

Handy Helper is a full stack Django MTV (MVC) job board application where users can list jobs or accept jobs that have been listed. The front end functionality is built using HTML, CSS, and Bootstrap. The back end logic is written in Python and uses the SQLite database. It is deployed on an Ubuntu 18.04 Azure Virtual Machine. HTTP requests to the application site are routed by an Nginx web server to a Gunicorn server that interfaces directly with the application.

Technologies used:

  • Django
  • SQLite
  • Azure VM
  • Nginx
  • Bootstrap
  • Python
  • HTML
  • CSS
